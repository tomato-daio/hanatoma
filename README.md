# はなとま (hanatoma)

AI英会話アウトプット練習PWA。シチュエーション別にAIと英語で会話し、発音（Azure・音素レベル）と文法・表現（Claude）の両面から精密に添削を受けられます。**毎日少しずつ続ける**ことを最優先に設計されています。

姉妹アプリ: [シャドとま (shadotoma)](../shadotoma) — インプット練習（シャドーイング）。はなとまはアウトプット練習（会話）を担当し、弱点音素・ストリークをデータ連携します。

## 機能

- **シナリオ会話**: 8カテゴリ×5レベルの40シナリオ（カフェ注文〜面接交渉）。キーフレーズ予習 → ガイド付き会話 → フリー会話 → 添削レポート
- **3つの練習モード**: フルレッスン（約10分）/ クイック会話（約5分）/ ひとくち英会話（1〜2分・忙しい日でもストリーク継続）
- **精密添削**: Azure発音評価（音素レベル・自由発話対応）+ Claude Sonnetによる文法・表現・自然さの添削とCEFR段階別リフレーズ
- **レベル診断と自動調整**: 初回診断でCEFRレベル判定、成績に応じて自動昇降格。AIの語彙・話速もレベル連動
- **ゲーミフィケーション**: XP・ランク・デイリークエスト・バッジ・シナリオ島マップ・週次ボス・お休みチケット付きストリーク
- **シャドとま連携**: 弱点音素からのシナリオ推薦・コンビストリーク・模範解答をシャドーイング教材として書き出し

## セットアップ（初回のみ）

1. アプリを開き、設定タブで以下の2つのキーを登録する（キーは端末内にのみ保存されます）
   - **Azure Speech**: Azureポータル → Speechリソース → キー1（無料枠F0で月5時間ぶん）
   - **Anthropic API**: console.anthropic.com → API Keys（会話・添削に使用。目安 月300〜500円）
2. ホームの案内からレベル診断テストを受ける（スキップも可能）
3. 「今日のレッスン」をはじめる

コスト暴走防止のため既定で1日3セッションの上限があります（設定で変更可）。使用量と概算金額は設定タブで確認できます。

## 開発

```
npm install
npm run dev          # http://localhost:5174
npm test             # Vitest（379件）
npm run build        # tsc + vite build
npm run validate-scenarios  # シナリオJSONのスキーマ検証
```

- 設計の正本: [DESIGN.md](DESIGN.md)（実装エージェントはこれに従うこと）
- 音声パイプラインの単体検証: `/#/selftest`（録音→発音評価→TTSを個別テスト）

## データの保存先

学習データ・録音・APIキーはすべて端末内（IndexedDB）。外部送信は Azure Speech（評価対象音声）と Anthropic API（会話テキスト）のみで、いずれも自分のAPIキーで直接送られます。バックアップ（設定→エクスポート）にAPIキーは含まれません。
